Lesson 4: PWM & Analog Output

What Is PWM?

Arduino cannot produce true analog voltage. Instead, it uses PWM (Pulse Width Modulation) to simulate analog output.

PWM works by turning a pin ON and OFF very fast. By changing how long the signal stays ON versus OFF, Arduino controls the average voltage.

PWM Pins on Arduino

Only pins with a ~ symbol support PWM.

  • ~3
  • ~5
  • ~6
  • ~9
  • ~10
  • ~11

Example: 

analogWrite(9, 128); // 50% brightness
