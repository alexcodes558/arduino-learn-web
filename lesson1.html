Lesson 1: Blink (The RIGHT Way)

What Are We Doing?

We are turning an LED on and off using an Arduino. But more importantly, we are learning what a pin actually is.

What Is a Pin?

A pin is like a tiny switch controlled by code. When the pin is HIGH, electricity flows. When it is LOW, electricity stops.

The Code

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

ðŸ”¹ HIGH = electricity ON

Common Mistakes

  • Forgetting pinMode()
  • Using the wrong pin
  • Thinking delay pauses the Arduino forever
