Lesson 1: Blink — How Pins Control Power

Concept First

An Arduino pin is not magic. It simply allows electricity to flow (HIGH) or stops it (LOW).

Interactive Diagram

Click the button to simulate the pin turning HIGH / LOW.

The Code Behind This

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
