An Arduino is very different.

An Arduino runs one program, and it runs it over and over forever.

There is no desktop. No apps. No stopping unless power is removed.

Every Arduino sketch has the same structure.

You do NOT need to understand the code yet. Right now, focus on the behavior.

ü§î Why This Matters

Many beginners think Arduino code runs from top to bottom once. That assumption causes almost every beginner mistake.

Arduino code is a behavior loop, not a script.

Once you truly understand this, everything else becomes easier.
