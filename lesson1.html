Lesson 1: Blink — Interactive

Concept First

An Arduino pin controls electricity. HIGH = ON, LOW = OFF. You will see this visually below.

Interactive Breadboard Diagram

Hover pins to see their function. Click pin to toggle HIGH/LOW.

The Code Behind It

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
