üì¶ Why Variables Matter

Without variables, Arduino would forget everything every moment.

Variables allow Arduino to:

  • Remember sensor readings
  • Track time
  • Make decisions

üî¢ Variable Types

Different boxes store different kinds of data.

int ledPin = 13; float voltage = 4.85; bool lightOn = true;

Choosing the correct type prevents bugs and saves memory.

üîÅ Variables and loop()

The key idea:

Variables keep their values between loop cycles.

loop() runs ‚Üí value changes ‚Üí loop() runs again ‚Üí value remembered
int count = 0; void loop() { count = count + 1; }

Each loop pass increases count. It does NOT reset.

üö´ Common Beginner Mistake

Declaring variables inside loop() when they need to persist.

Those variables are recreated each cycle.
