Lesson 5: Analog Input

What Is Analog Input?

Unlike digital pins, analog pins do not read ON or OFF. They read variable voltages.

Arduino converts voltage into numbers using an Analog-to-Digital Converter (ADC).

Arduino Analog Pins

Analog pins measure voltages between 0V and 5V.

Potentiometer (Variable Resistor)

A potentiometer lets us control voltage manually. It has three pins:

  • 5V
  • Signal (to A0)
  • GND

Interactive Analog Reading

Move the slider to simulate turning the potentiometer.

Analog Read: 512

Arduino reads analog values from 0 to 1023.

Arduino Code

int potPin = A0;
int value;

void setup() {
  Serial.begin(9600);
}

void loop() {
  value = analogRead(potPin);
  Serial.println(value);
  delay(200);
}

The ADC converts voltage into a number you can use in code.
