1. How Arduino Code Executes

Arduino programs always follow the same structure. The setup() function runs exactly one time when the board powers on. The loop() function runs continuously forever. This allows Arduino to constantly react to sensors and inputs.

void setup() { pinMode(13, OUTPUT); } void loop() { digitalWrite(13, HIGH); }

5. Functions & Loops (Real Arduino Style)

Functions allow you to reuse logic. Loops repeat logic efficiently.

void blinkLED(int times) { for (int i = 0; i < times; i++) { digitalWrite(13, HIGH); delay(500); digitalWrite(13, LOW); delay(500); } }
