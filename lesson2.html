Lesson 2: Buttons & Input

Learning Section

In this lesson, we will learn how to use buttons with Arduino and understand how inputs and outputs work. You will learn about:

  • Input vs Output: Inputs read signals from sensors or buttons. Outputs control LEDs, motors, and other devices.
  • HIGH vs LOW: HIGH means the pin is receiving voltage, LOW means 0 volts.
  • Pull-up / Pull-down resistors: These resistors prevent pins from "floating," which can cause random readings.
  • Button bouncing: Pressing a button can cause the pin to rapidly switch HIGH/LOW before stabilizing.
  • Logic Basics: OR logic: LED lights if any button is pressed. AND logic: LED lights only if all buttons are pressed.

Code Example

void setup() {
  pinMode(2, INPUT);
  pinMode(3, INPUT);
  pinMode(12, OUTPUT);
  pinMode(13, OUTPUT);
}

void loop() {
  if(digitalRead(2)==HIGH || digitalRead(3)==HIGH) digitalWrite(13,HIGH);
  if(digitalRead(2)==HIGH && digitalRead(3)==HIGH) digitalWrite(12,HIGH);
  else digitalWrite(12,LOW);
}
