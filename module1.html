How Arduino Code Works

Lesson Reading

Arduino programs are written in a simplified version of C++. Every Arduino sketch follows the same structure.

There are only two required functions:

  • setup() – runs once when the board powers on
  • loop() – runs forever, over and over
void setup() { // runs once } void loop() { // runs repeatedly }

The Arduino does not stop running unless power is removed. When loop() finishes, it immediately starts again.
